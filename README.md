# Employee Management System (GUI Version)

## Top Features:

1. **Add User**
2. **Show User**
3. **Delete User** (Admin Passcode: `1234`)
4. **Exit**

## Usage:

1. Clone the repository to your local machine.
2. Navigate to the directory.
3. Double click on the "Swing Project.jar" file.
